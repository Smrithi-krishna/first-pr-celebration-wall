# 🚀 First PR Celebration Wall

A beginner-friendly open-source project where developers make their **first Pull Request (PR)** and get featured on a live celebration wall 🎉

---

## ✨ About

This project helps beginners understand:
- Git & GitHub basics
- How Pull Requests work
- Open-source contribution flow

Every contributor adds their card in `index.html` and submits a PR 🚀

---

## 🌍 How It Works

1. Fork this repository  
2. Clone your fork  
3. Open `index.html`  
4. Add your PR card inside the wall section  
5. Commit and push changes  
6. Create a Pull Request 🎉  
7. Get featured on the wall

---

## 🧱 Tech Stack

- HTML5  
- CSS3 (Glassmorphism UI)  
- Vanilla JavaScript  

---
