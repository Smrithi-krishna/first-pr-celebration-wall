How it works

Four steps. Zero gatekeeping. Welcome to open source.

01

Fork the repo

Click Fork on GitHub to create your own copy of the project under your account.

02

Clone & edit

git clone your fork, then add yourself to contributors.json or the wall file.

03

Commit & push

Stage your change, write a meaningful commit, and push it to a new branch on your fork.

04

Open a Pull Request

Open a PR back to the main repo. Once merged, your card appears on the wall ✨
